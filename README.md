# Google News CLI

A simple and straight forward terminal CLI application to fetch and search Google News.

## Installation

You can install the dependencies by running:

```bash
npm install
```

To install it globally so you can use the `gnews` command anywhere, run:

```bash
npm install -g .
```

## Usage

If installed globally, you can simply run:

```bash
gnews
```

Alternatively, you can run it via npm:

```bash
npm start
```
